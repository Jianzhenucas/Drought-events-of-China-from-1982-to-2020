# Drought-events-of-China-from-1982-to-2020

File naming: XXXX-Area.mat, XXXX-Centroid.mat, XXXX-Distance.mat, and XXXX-Severity.mat. Here, XXXX represents different drought types: MD, SD1/SD2/SD3/SD4, and VD denote meteorological drought, soil drought at different depths, and vegetation drought, respectively; EMD, ESD1/ESD2/ESD3/ESD4, and EVD denote extreme meteorological drought, extreme soil drought at different depths, and extreme vegetation drought, respectively.

The file XXXX-Area.mat contains the range characteristics affected by different drought events: the first column is the ID of the drought event; the second and third columns represent the start date (year.month) of the drought event, respectively; the fourth column indicates the duration of each drought event, measured in months; the fifth column represents the area affected by the drought event, in km².

The file XXXX-Centroid.mat contains the location characteristics of different drought events for each month: the first column is the ID of the drought event; the second column represents the occurrence date (year.month) of the drought event; the third and fourth columns represent the longitude and latitude coordinates of the centroid of the drought event, respectively, in degrees.

The file XXXX-Distance.mat contains the migration characteristics of all drought events for a specific drought type: the first column represents the ID of the drought event; the second column represents the migration distance of the drought event's center, in km; the third and fourth columns represent the start time (year.month) of the drought event, respectively; the fifth column represents the duration of the drought event, measured in months.

The file XXXX-Severity.mat contains the characteristics of the impact degree of all drought events: the first column represents the ID of the drought event; the second and third columns represent the start time (year.month) of the drought event, respectively; the fourth column represents the duration of the drought event, measured in months; the fifth column represents the severity of the drought event, in month·km².
